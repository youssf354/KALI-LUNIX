Kali Linux Tools

Discover the most important penetration testing and information security tools

Nmap

Network scanning and security analysis tool. Used to scan devices and services available on the network.

Basic Commands:

nmap -sS -sV [IP]    # Quick service scan
nmap -p- [IP]         # Scan all ports
nmap -O [IP]          # OS detection

Wireshark

Network packet analyzer. Used for analyzing network traffic and troubleshooting network issues.

Basic Usage:

wireshark -i [interface]    # Start packet capture
wireshark -r [file]           # Open capture file

Burp Suite

Integrated framework for web application security testing. Includes tools for analyzing and diagnosing vulnerabilities.

Key Features:

  • Proxy for traffic control
  • Scanner for vulnerability detection
  • Intruder for vulnerability testing

OWASP ZAP

Open-source tool for web application security testing. Helps in discovering security vulnerabilities.

Basic Usage:

zap-cli quick-scan [URL]    # Quick scan
zap-cli active-scan [URL]     # Active scan

Aircrack-ng

Suite of tools for wireless network security testing. Used for analyzing and penetrating Wi-Fi networks.

Basic Commands:

airmon-ng start wlan0    # Enable monitor mode
airodump-ng wlan0mon       # Capture packets
aircrack-ng [file]        # Crack encryption

Autopsy

Graphical interface for digital forensics analysis. Used for examining devices and extracting evidence.

Key Features:

  • Image analysis
  • Deleted file recovery
  • System log analysis
